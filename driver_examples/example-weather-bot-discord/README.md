# Weather Bot for Discord
This is a Membrane program that provides weather information. Use the command `/weather city <name>` in discord, the bot will return the current weather for the given city.

## How to use

Paste the following URL in your browser to open the directory inside the Membrane VSCode Extension.

vscode://membrane.membrane/directory/example-weather-bot-discord

Run `:configure` and set the `guildId` aka (channel).
